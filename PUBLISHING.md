# 🚀 Cadet Stationery - Deployment Guide

Mubarak ho! Aap ki $100,000 premium website live hone ke liye tayyar hai. Isay GitHub Pages par upload karne ke liye ye steps follow karein:

---

## 1️⃣ GitHub par Account aur Repository
1. [GitHub.com](https://github.com) par jayein aur login karein.
2. Naya repository banayein (+ icon -> New repository).
3. Repository Name: `cadet-stationery`
4. Isay **Public** rakhein.
5. "Create repository" par click karein.

## 2️⃣ Files Upload Karein
1. Apne repository ke page par **"uploading an existing file"** par click karein.
2. Apne computer se `stationery-store` folder ki saari files select karein aur GitHub par drag-and-drop kar dein.
3. "Commit changes" par click karein.

## 3️⃣ Website Live Karein (GitHub Pages)
1. Repository ki **Settings** tab mein jayein.
2. Left side par **Pages** menu par click karein.
3. "Branch" section mein `main` ya `master` select karein aur folder `/ (root)` choose karein.
4. **Save** par click karein.

## 4️⃣ Aap ka Link
2-3 minute baad aap ki website is link par live ho jaye gi:
`https://YOUR-USERNAME.github.io/cadet-stationery/`

*(Note: `YOUR-USERNAME` ki jagah apna GitHub username likhein)*
